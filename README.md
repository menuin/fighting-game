# Javascript Fighting Game

<img src="https://github.com/menuin/fighting-game/assets/63971484/a9dfdd86-d819-48ce-bf2d-e48feb9d4160" width=900/>

### Player 1
- `a` : move backward <br/> 
- `w` : jump <br/>
- `s` : attack <br/>
- `d` : move forward <br/>

### Player 2
- `←` : move backward <br/> 
- `↑` : jump <br/>
- `↓` : attack <br/>
- `→` : move forward <br/>
